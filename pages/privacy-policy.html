Privacy Policy

Informativa sul trattamento dei dati personali ai sensi del Regolamento UE 2016/679 (GDPR)

Titolare del trattamento

IISS Giudici Saetta e Livatino
Il presente sito web e' un progetto didattico realizzato nell'ambito del percorso di potenziamento pomeridiano della classe 5A SIA, sotto la supervisione del Prof. Enrico Maria Caruso.

Tipologia di dati raccolti

Questo sito non raccoglie dati personali in modo diretto. Non sono presenti moduli di contatto, sistemi di registrazione, aree riservate, sistemi di commento ne' strumenti di analisi del traffico.

Dati di navigazione

Come tutti i siti web, i server di GitHub Pages (che ospita il sito) potrebbero registrare automaticamente alcuni dati tecnici durante la navigazione, tra cui: indirizzo IP, tipo di browser, sistema operativo, data e ora della richiesta. Questi dati non vengono trattati dal titolare e sono gestiti esclusivamente da GitHub secondo la propria informativa privacy.

Dati archiviati localmente

Il sito utilizza il localStorage del browser per salvare:

  • La preferenza del tema (chiaro/scuro)
  • Lo stato di accettazione del banner cookie

Questi dati rimangono esclusivamente sul dispositivo dell'utente e non vengono trasmessi ad alcun server.

Servizi di terze parti

Google Fonts

Il sito carica il font "Inter" tramite il servizio Google Fonts. Ad ogni visita, il browser effettua una richiesta ai server di Google, che potrebbe comportare il trasferimento di dati tecnici (indirizzo IP). Per approfondimenti: Privacy Policy di Google.

GitHub Pages

Il sito e' ospitato sulla piattaforma GitHub Pages di GitHub Inc. Per informazioni sul trattamento dei dati da parte di GitHub: GitHub Privacy Statement.

Base giuridica del trattamento

Il trattamento dei dati tecnici necessari al funzionamento del sito si basa sul legittimo interesse (art. 6, par. 1, lett. f del GDPR) a garantire la corretta erogazione del servizio. L'utilizzo del localStorage per le preferenze utente si basa sul consenso espresso tramite il banner cookie.

Trasferimento dati extra-UE

I servizi di terze parti utilizzati (GitHub, Google) potrebbero trattare i dati su server situati al di fuori dell'Unione Europea. Tali trasferimenti avvengono in conformita' con le garanzie previste dal GDPR (Capo V), incluse le Clausole Contrattuali Standard (SCC) e le certificazioni previste dal Data Privacy Framework UE-USA.

Diritti dell'utente

Ai sensi degli artt. 15-22 del GDPR, l'utente ha diritto di:

  • Accedere ai propri dati personali
  • Richiedere la rettifica o la cancellazione dei dati
  • Limitare od opporsi al trattamento
  • Richiedere la portabilita' dei dati
  • Revocare il consenso in qualsiasi momento
  • Proporre reclamo all'Autorita' Garante per la protezione dei dati personali (www.garanteprivacy.it)

Trattandosi di un sito statico che non raccoglie dati personali, la maggior parte di questi diritti si applica esclusivamente ai dati eventualmente trattati dai servizi di terze parti sopra indicati.

Modifiche alla presente informativa

La presente informativa puo' essere aggiornata in qualsiasi momento. Le modifiche saranno pubblicate su questa pagina con indicazione della data di ultimo aggiornamento: marzo 2026.
